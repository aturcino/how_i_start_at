# HowIStartAt

HowIStartAt is simple gem to show how I begin Ruby project.

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'how_i_start_at'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install how_i_start_at
